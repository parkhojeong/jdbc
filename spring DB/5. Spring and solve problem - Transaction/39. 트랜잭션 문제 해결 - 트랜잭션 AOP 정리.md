Transaction AOP flow
1. call AOP Proxy
2. Proxy get TransactionManager from Spring Container
3. TransactionManager.getTransaction
4. get connection using DataSource
5. connection.setAutoCommit(false) - start transaction
6. store connection in TransactionSynchronizationManager
7. TransactionSynchronizationManager manage connection

* business logic
8. call business logic
9. repository get connection from TransactionSynchronizationManager
