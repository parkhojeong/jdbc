* DataSource auto registration
- if dataSource is not registered, spring boot registry DataSource
 - default HikariDataSource 
- bean name: dataSource

spring boot set from application.properties
if url is not set, memory db is used
[application.properties]
spring.datasource.url=
spring.datasource.username=
spring.datasource.password=

* TransactionManager auto registration
- if transactionManager is not registered, spring boot registry transactionManager
 - default is determined from library list
   - JDBC -> DataSourceTransactionManager
   - JPA -> JpaTransactionManager
 - bean name: transactionManager
